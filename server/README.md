AISLE EARTHLINGS 

This application has a Django backend tied with a Postgres database and a React frontend. Utilizing Nginx as a proxy server and gunicorn to handle backend loads efficently. The instacart api will be used to locate stores and their inventory near you and gemini api will be used to gather prices estimates for products aswell 

A comprehensive compendium gorcery store application for vegans who want to easily locate and compare vegan grocery item pricing and avilablity near them.
Aisle Earthlings is unlike other grocery store applications because it filters out what vegan dont need to or care to see, it always provides pictures of nutrition facts so you can double check its veganibility and will let you side by side compare stores which other grocery apps fail to allow for mindful shopping
